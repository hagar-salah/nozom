أنواع البيانات في GIS
Vector Model — Raster Model — Geodatabases
▾
Vector Model
📐 النموذج الاتجاهي
يستخدم النقاط والخطوط والمضلعات. مثالي لتمثيل الكيانات المنفصلة مثل المباني والطرق.
Raster Model
🔲 النموذج الشبكي
يتكون من شبكة من الخلايا (Pixels). مثالي لتمثيل الظواهر المستمرة مثل الارتفاعات ودرجة الحرارة.
Geodatabases
🗄️ قواعد البيانات الجغرافية

  • النقاط: مثل مواقع الآبار
  • الخطوط: مثل الطرق والأنهار
  • المضلعات/المساحات: مثل حدود الدول أو الغابات
  • البيانات الشبكية: مثل صور الأقمار الصناعية

SECTION 04 • رابعاً
وظائف نظم المعلومات الجغرافية الأربع
إدخال — تخزين — معالجة وتحليل — إخراج
▾
١
إدخال المعلومات إلى النظام
أول وظيفة وأصعب المراحل وأكثرها عرضة للخطأ. تكلفتها قد تصل لـ 70% من التكلفة الإجمالية. تشمل: جمع المعلومات + التأكد من صحتها + تحريرها وتحليلها + استيرادها وتصديرها. المصادر: خرائط، مسح ميداني، صور جوية، استشعار عن بُعد.
٢
تخزين المعلومات في النظام
من أهم مميزات GIS تخزين المعلومات في طبقات. سهل دمج المعلومات لكن صعب تفريق المركّبة. أنواع التخزين: أساسي + مؤقت + نسخ احتياطية. التخزين المكاني (خطي) أكثر إتاحةً للتحليل من الشبكي.
٣
معالجة وتحليل المعلومات
أهم العمليات: الوظائف الكارتوجرافية (تغيير مقياس، تحويل صيغ، تغيير مسقط، إضافة مفتاح ورموز) + الاستعالم والبحث (أفضل وظيفة عرفت بها GIS) + قياس المعالم + التحليل الإحصائي.
٤
إخراج النتائج
خرائط ورقية ورقمية + الخرائط التفاعلية على الويب (WebGIS) + التمثيل ثلاثي الأبعاد (3D GIS) + جداول + تقارير + صفحات الإنترنت.

SECTION 06 • سادساً
علم المعلومات المكانية Spatial Information Science
المفهوم + المجالات الخمسة + الأهمية + العلاقة بالعلوم الأخرى
▾

علم حديث متعدد التخصصات يهتم بجمع ومعالجة وتحليل وتمثيل وإدارة المعلومات المرتبطة بالمكان والحيز الجغرافي. يدرس البيانات ذات الموقع الجغرافي ويطور الأساليب والتقنيات لفهم الأنماط والعلاقات المكانية.

المجالات الخمسة
١
📍 البيانات المكانية
نقاط، خطوط، مضلعات، بيانات شبكية.
٢
🛰️ تقنيات جمع البيانات
الاستشعار عن بُعد + GPS + المسح الجغرافي التقليدي والرقمي.
٣
🗄️ إدارة قواعد البيانات المكانية
تخزين في GIS + Big Data + الذكاء الاصطناعي.
٤
📊 التحليل المكاني
دراسة الأنماط والتجمعات + النمذجة المكانية + التحليل الشبكي.
٥
🗺️ التمثيل البصري
إنتاج الخرائط الرقمية والتفاعلية + النمذجة ثلاثية الأبعاد والواقع الافتراضي.
علاقة SIS بالعلوم الأخرى
  • GIS: الأداة الرئيسية لتطبيق علم المعلومات المكانية
  • التحليل المكاني: فرع تطبيقي يستخدم مبادئ SIS لفحص الأنماط والعلاقات
  • الاستشعار عن بُعد: مصدر مهم للحصول على البيانات المكانية
  • علوم البيانات والذكاء الاصطناعي: تُستخدم لمعالجة الكم الهائل من البيانات المكانية

📌 ملخص المحاضرة الأولى — أبرز النقاط

GIS vs GIScience: GIS أداة عملية. GIScience جانب علمي ونظري أوسع نطاقاً.
٥ مكونات: بيانات جغرافية + أجهزة + برمجيات + أفراد + إجراءات وسياسات.
٣ أنواع بيانات: Vector (نقاط/خطوط/مضلعات) + Raster (خلايا) + Geodatabases.
٤ وظائف: إدخال (أصعب وأغلى مرحلة 70%) ← تخزين ← معالجة وتحليل ← إخراج.
الاستعالم والبحث: أفضل وظيفة عرفت بها GIS — يجمع شروط مكانية ووصفية معاً.
الاتجاهات الحديثة: AI + Cloud GIS + Big Data + WebGIS + 3D/4D + IoT.

١. التخطيط العمراني: اختيار مواقع المدارس والمستشفيات.
٢. البيئة: مراقبة التصحر وتآكل التربة.
٣. الصحة العامة: رسم خرائط انتشار وباء معين.
٤. النقل: تحديد أفضل مسارات الطرق.
٥. إدارة الكوارث: خرائط الفيضانات والزلازل.
س ١٩
مقالي
اشرح الوظائف الأربع لنظم المعلومات الجغرافية مع ذكر أهم ما يميز كل وظيفة.
💡 ١ — الإدخال: أول وأصعب المراحل وأكثرها عرضة للخطأ، تكلفتها قد تبلغ 70% من الإجمالية. تشمل: جمع البيانات + التحقق من صحتها + تحريرها + استيرادها.

٢ — التخزين: يتم في طبقات وهو أهم مميزات GIS — سهل الدمج صعب التفريق — التخزين الخطي أكثر إتاحةً للتحليل من الشبكي.

٣ — المعالجة والتحليل: تشمل الوظائف الكارتوجرافية + الاستعلام والبحث (أفضل وظيفة GIS) + قياس المعالم + التحليل الإحصائي.

٤ — الإخراج: خرائط ورقية ورقمية + WebGIS + 3D GIS + جداول وتقارير وصفحات إنترنت.
س ٢٠
مقالي
ما المجالات الخمسة لعلم المعلومات المكانية (SIS) وكيف يرتبط بنظم المعلومات الجغرافية؟
💡 المجالات الخمسة:
١. البيانات المكانية (نقاط، خطوط، مضلعات، شبكية)
٢. تقنيات جمع البيانات (استشعار عن بُعد + GPS + مسح جغرافي)
٣. إدارة قواعد البيانات المكانية (GIS + Big Data + AI)
٤. التحليل المكاني (أنماط + نمذجة + تحليل شبكي)
٥. التمثيل البصري (خرائط رقمية + 3D + واقع افتراضي)

علاقته بـ GIS: GIS هو الأداة الرئيسية لتطبيق SIS. التحليل المكاني فرع تطبيقي يستخدم مبادئ SIS. SIS أشمل وأوسع بينما GIS جزء منه يركز على النظم الحاسوبية.
